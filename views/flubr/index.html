Flubr back end stack

  • Node.js - low-level javascript networking platform
  • Connect - middleware framework (analagous to Rack)
  • Express - lightweight & minimal server framework (analagous to Sinatra)
  • Mongo - document storage database (NoSQL)

Flubr front end stack

  • Stylus & Nib - CSS abstraction language (analagous to SASS & Compass)
  • Jade - HTML abstraction (analagous to HAML)
  • Coffeescript

Flubr tooling

  • Gulp - javascript task runner (analagous to Rake)
  • Browserify - front-end module loader
